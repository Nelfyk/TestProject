Приложение TestProject.

- Название БД - shop.
- Дамп БД лежит в корне проекта c названием "shop.tar".
- Сборка проекта осуществлена через maven (mvn install)
- 4 входных файла с запросами search(search1.json, search2.json и т.д.) и два с запросами stat (stat1.json,stat2.json).
- Файл "config.ini" в котором прописываются данные для подключения к базе(хост, порт, логин и т.д.).
- Всё эти файлы копируются в target/classes при сборке через maven. Туда и обращается программа.
- В папку target собирается TestProject.jar для запуска

- Любая ошибка записывается в выходной файл формата json.
